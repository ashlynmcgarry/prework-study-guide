HTML

  • The head element contains information about the webpage.
  • The body element represents the visibile content shown to the user.
  • Child elements are nested in other elements
  • HTML is the frame, CSS is the styling, Javascript is the interactivity.
  • UHF-8 is most characters when selecting language.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS applies to a specific HTML tag using a style attribute with a CDD rule to style a particular element.
  • Internal style sheet adds CSS rules in the head of the HTML file and only applies to all tagged elements on that page.
  • External style sheet applies CSS rules in a seperate file, makes changes easier.
  • Seperation of concerns: each section of code has its own responsibility.
  • Selector: element to which a declaration will apply
  • Declaration: property and value.
  • *: applies rule to all page elements
  • DRY: do not repeat yourself, reuse or share code to minimise number of lines.
  • Block: assigns each img to start on new line.
  • rel: specifies relationship between doc & linked resource.
  • href: specifies location (URL), derived from current location.

Git

  • git status: checks what brance we're currently on
  • git checkout -b branch-name: creates a new brance and switches to it
  • git checkout main: switches back to main branch
  • Repository/repo: A folder for a project that contains all files and history
  • Branch: workspace where you make changes that dont affect the live site
  • Markdown: a way to write for web uing plain text
  • Commit changes: saving a new version of a file. Add, edit or deleting a file is committing.
  • Pull: pulls branch modifications to local environment.
  • Push: pushes branch modifications to remote environment

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • ; indicates where a statement ends, it's good practice to add it at the end of each statement.
  • JavaScript is case sensitive.
  • String is a sequence of text enclosed in "" or ''.
  • Numbers don't have quotation marks.
  • Boolean values are true or false, no quoatations marks needed.
  • An array allows you to store multiple values in a single reference.
  • Everything in JavaScript is an object, an object can be anything.
  • Comments are marked with */ or // and are ignored by the browser.
  • Operators are a math symbol that produces a result based on two values or variables and include +, -, * (multiplication) or / (division).
  • Strict equality === tests if two values are equal and of the same type, returns a boolean response.
  • Interactivity requires Event handlers, they look for activity in the browser and run code in response.
  • Null type is exactly one value 'null', the absense of an object.
  • Undefined type is exactly one value 'undefined', is the absense of a value.
  • Variable is a named container to store data in code.
  • const: variables that cannot be reassigned a value, locks down what the variable will be.
  • Variable names must be unique.
  • =: Assignment operator that assigns data to be stored in a variable.
  • To access a string refer to the variable name.
  • Falsy is 0, -0, 0n, " ", null, NaN, undefined or false, everything else is Truthy.
  • console.log: Output a message to the web console by adding an argument.
  • Control flow: Order that a computer executes code in a script, JS runs from the first statement at the top to the last statement at the bottom.
  • if(condition): executes a block of code if the condition is true.
  • Arrays hold a group of data which is often related in some way.
  • Index: Allows us to work with individual peices of data in an array by showing the location of that data within the array.
  • First item of an array is always 0 not 1.
  • For loop: Uses the predictable patterns of indecies to perform a task on all array items.
  • To use a for loop you need an index and number of arrays.
  • let: allows reassignment of a variable over time.
  • A for loop contains: A starting point, a condition and a statement that allows the array to iterate.
  • Function: a set of instructions that tells a computer how to perform a task.
  • To execute a function, first define the function then call the function.
  • Functions are used to define and reuse blocks of code.
